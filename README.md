# Innovation Diffusion Analysis: Pentax 17
## Marketing Analytics Project - Bass Diffusion Model

**Project Overview:** This project analyzes the potential market diffusion of the Pentax 17, a half-frame film camera released in 2024. By using historical data from the "Golden Age" of 35mm film cameras (1977–2007), I estimated the coefficients of innovation (_p_) and imitation (_q_) to forecast how this modern revival product will be adopted globally over the next 20 years.


### **Directory Structure**

📁 img/
This directory contains all the visualizations used in the analysis.
- image1.png (Historical Film Camera Diffusion): A plot showing the actual CIPA historical shipment data overlaid with the Bass Model's fitted curve to demonstrate the model's accuracy in capturing historical trends.
- image2.png (Pentax 17 Predicted Diffusion Path): A 20-year diffusion projection for the Pentax 17, visualizing annual sales (bar chart) and cumulative adopters (line graph).

📁 data/
This directory holds the datasets used for the analysis.
- film_cameras.xlsx: The primary historical dataset containing global 35mm film camera shipments (1977-2007) sourced from CIPA.
- pentax_predictions.csv: The output table generated by the model, containing the year-by-year forecast of annual and total adopters for the Pentax 17.

📁 report/
This directory contains the project report and its source files.
- report.pdf: The final project report including the executive summary, look-alike justification, and strategic conclusions.
- report_source.md: The Markdown source code used to generate the final report.

📄 Root Directory (/)
The root directory holds the execution files for the project.
- bass_film_cameras.ipynb: The main interactive notebook containing the full data cleaning, visualization, and table generation.
- script1.py: Python script used to load the historical data and estimate the parameters (_p, q, M_).
- script2.py: Python script used to run the Pentax 17 forecast and export the final prediction table.
- helper_functions.py: A collection of helper functions, including the core Bass Model mathematical formula used by both scripts.


### **Methodology Summary** 
1. Look-alike JustificationI selected Historical 35mm Film Camera Sales as the "look-alike" for the Pentax 17 because both represent the same mechanical and chemical technology. By analyzing the 1977–2007 era, we can model the social contagion (imitation) inherent to analog photography.
2. Parameter EstimationBased on the historical fit, the following parameters were derived: _p_ (Innovation): 0.010203, _q_ (Imitation): 0.162198, _M_ (Market Potential): 707,311 (units in 1000s).
3. Fermi Logic for Pentax 17To adapt the model for a 2025 niche product, I estimated a modern Market Potential ($M$) of 5,000,000 units (5,000 in the model) based on a global community of 20 million enthusiasts and a 25% target niche market share.
