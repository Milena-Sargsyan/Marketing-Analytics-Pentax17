# Homework 1: Bass Diffusion Model - Pentax 17
Course: DS 223 | Marketing Analytics

Student: Milena Sargsyan

## Project Overview
Analysis of the potential market diffusion for the Pentax 17 (2025). This project uses historical CIPA shipment data to estimate adoption parameters.

## Directory Structure
- **docs/**: Contains the source reference `Shipment of Film Cameras.pdf` (CIPA).
- **data/**: 
    - `film_cameras.xlsx`: Historical data (1977-2007) manually extracted from the CIPA report.
    - `pentax_predictions.csv`: 20-year adoption forecast generated by the model.
- **img/**:
    - `image1.png`: Historical fit of the Bass Model to CIPA data.
    - `image2.png`: Predicted adoption path for the Pentax 17.
- **report/**:
    - `report.pdf`: The final project report.
    - `report_source.md`: Markdown source file for the report.
- **Root (Bass_Model/)**:
    - `script1.py`: Parameter estimation script.
    - `script2.py`: Forecast and CSV export script.
    - `helper_functions.py`: Mathematical core of the Bass Model.
    - `bass_film_cameras.ipynb`: Full analysis notebook.
    - `requirements.txt`: List of Python dependencies.

## Data Source & Credibility
The historical data used for this analysis was manually extracted from the **CIPA (Camera & Imaging Products Association)** "Shipment of Film Cameras" report. A PDF copy of this source is available in the `docs/` directory of this repository for academic verification.
